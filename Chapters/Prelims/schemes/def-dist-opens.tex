\begin{definition}[Distinguised open]\label[definition]{Distinguised open}
Let $\Spec R$ be a affine scheme.
The set \[D(f)=\set{\Ip \subset R | f\not\in\Ip}\] for a global section $f$ is called a distinguised open.
The open $D(f)$ is isomorphic to $\Spec{R_f}$ as a locally ringed space.%todo: ref
\end{definition}
