\begin{definition}[Sections functor]\label[definition]{728}
For any $a\in \cat{C}$ define the functor 
\[\sections{a}: \pshs{C} \rightarrow \Sets\]
by
\[\presheaf{F} \rightarrow \psh{F}(a).\]

For any presheaf $\psh{G}$, we define
\[ \sect{\psh{G}} : \pshs{C} \rightarrow \Sets,\]
\[ \psh{F} \rightarrow \Hom{\psh{G}}{\psh{F}}.\]

We will mostly use this for the terminal presheaf, 
which will allow us to compute the global sections.
\end{definition}
