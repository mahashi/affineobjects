Let $Y = \ringedsite{X}{T}{O}$ be a ringed site.
Let $R = \globalsections{O}$.

A presheaf module on $Y$  is a presheaf of sets $\presheaf{F}$ on $\cat{X}$ together with a map of presheaves
\[ \presheaf{O} \times \presheaf{F} \rightarrow \presheaf{F}\]

such that for every object $a \in X$ the map $\sections{a}{O} \times \sections{a}{F} \rightarrow \sections{a}{F}$
defines a $\sections{a}{O}$-module structure on $\sections{a}{F}$.

A morphism $$\presheaf{F} \rightarrow \presheaf{G}$$ is a morphism of presheaf modules if 

\begin{center}
	\begin{tikzpicture}[]
		\node (lu) {$\presheaf{O} \times \presheaf{F}$}
		\node (ru) [right of = lu] {$\presheaf{F}$}
		\node (lb) [below of = lu] {$\presheaf{O} \times \presheaf{G}$}
		\node (rb) [right of = lb] {$\presheaf{G}$}

		\draw[->] (lu) to node (ru)
		\draw[->] (lu) to node (lb)
		\draw[->] (lb) to node (rb)
		\draw[->] (ru) to node (rb)
	\end{tikzpicture}
\end{center}
commutes.
The category of presheaf modules on $Y$ will be denoted $\pshmodules{Y}$.
